Namespace LrSlideshow

This namespace provides access to Slideshow commands.

Access the functions directly from the imported namespace.

Summary

LrSlideshow.startSlideshow()
Begins an impromptu slideshow using the photos currently in the filmstrip.
LrSlideshow.stopSlideshow()
Stops a running slideshow.
